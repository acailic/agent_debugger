# ADR-001: Product Positioning

**Status:** Under Review
**Date:** 2026-03-23

## Open Challenge

"Decision trees" as hero positioning may be too technical for the target audience. Users may respond more to "replay" or "monitoring" as the lead value prop. Needs user validation before committing positioning.

**Action:** Test 2-3 positioning variants with early beta users before finalizing.

---

## Original Decision (Deferred)

Position as the specialized agent debugger, not a general LLM observability platform.

Core differentiators proposed:
1. Decision-tree debugging
2. Evidence provenance
3. Safety/refusal observability
4. Time-travel replay
5. Multi-agent debugging
6. Real-time anomaly detection

See ADR-010 for related challenge: too many differentiators, should pick 1-2 hero features.
